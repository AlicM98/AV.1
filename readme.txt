🖌️Creative Studio - Site para Designer Gráfico

Este é um site desenvolvido para apresentar o portfólio e os serviços de um *Designer Gráfico*, focado em transmitir criatividade, profissionalismo e inovação. O objetivo é oferecer uma experiência visual impactante, destacando projetos, habilidades e valores que conectem o designer aos seus clientes de forma autêntica.

🎨 Recursos do Site 
- Página Inicial: Apresentação do designer com um visual moderno e elegante.  
- Portfólio: Exibição de projetos realizados, com detalhes sobre cada criação.  
- Serviços: Descrição dos serviços oferecidos, como branding, identidade visual, criação de logotipos e mais.  
- Sobre: Biografia do designer, trajetória profissional e missão.  
- Contato: Formulário para clientes entrarem em contato facilmente.

💻 Tecnologias Utilizadas  
- HTML5 
- CSS3  
- JavaScrip
